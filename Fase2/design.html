Entorno teorico de las etiquetas básicas de HTML

la tematica principal de nuestro portal es dar una teoria al usuario sobre las etiquetas básicas del lenguaje HTML como funcionan junto a sus atributos y valores. Para dar conocimiento de la impotancia de estas al crear, modificar y conocer los valores y caracteristicas de estas etiquetas.

HTML

HTML es un Markup Language, lo que significa que esta escrito con codigos que pueden ser leidos por una persona sin ser necesario que compilen estos codigos primero. Es decir el texto en una pagina web esta "Señalado" con estos codigos para dar instrucciones al navegador web sobre como mostrar el texto. Para "Señalar" el texto se necesitan etiquetas propias de HTML como son algunas de ellas: body - Para contenido / head - información sobre el documento / p - Para escribir un parrafo / h1...h6 - Para titulos.
